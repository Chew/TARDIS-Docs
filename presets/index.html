Chameleon Presets

Chameleon Presets can be selected and applied in the Chameleon Circuit GUI. There are 64 presets in total on two pages, plus one special hidden preset, and one custom preset that server admins can set up themselves — see the Custom preset page.

There are also some adaptive presets that are selected by biome when the Chameleon Circuit is set to Adaptive Biome mode. You can view those on the Adaptive Presets page.

Page one​

These presets use regular Minecraft blocks. When materialising they cycle through stained glass and regular glass for their animation.
