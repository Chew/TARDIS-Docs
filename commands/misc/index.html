Miscellaneous commands

Some utility commands available in the TARDIS plugin.

/tardis?​

Get help with TARDIS plugin commands.

CommandAliasesPermission
/tardis?t?, tardishelpnone needed
Get help with TARDIS commands.
/tardis? [command] [argument]

/tardisartron​

You can transfer energy either from your TARDIS, or using your own Time Lord energy. To do this you hold an Artron Storage Cell in your hand and use the /tardisartron command. For more information see Artron Storage Cells.

CommandAliasesPermission
/tardisartrontartrontardis.store
Transfer Artron Energy to an Artron Storage Cell
/tardisartron [tardis|timelord] [amount]

/tardissay​

Send translated text in chat. For more information see TARDIS Universal Translator.

CommandAliasesPermission
/tardissaytsaytardis.translate
Say something in a different language via the TARDIS Universal Translator.
/tardissay [language] [message]

/tardisgravity​

This command will let you create either sideways gravity wells, or up and down ones of any height (though you’ll need to build your own gravity shaft). For more information see Gravity Wells.

CommandAliasesPermission
/tardisgravitytgravitytardis.gravity
Add or remove gravity well blocks.
/tardisgravity [up|down|north|west|south|east|remove] [distance] [velocity]

/tardiscall​

Request a player to bring their TARDIS to your targeted location.

TARDIS call

If you're wandering around in your TARDIS, and a friend wants you to come and check something out, rather than them saying in chat "Hey come check this out" and then you using /ttravel [player] ? to request to travel to them (and them possibly having to move to a safe landing spot and accepting) - they can just run the command tardiscall [player] and you'll get a message saying "{Player} wants you to travel to them, do you accept?" which you accept to set the destination.

CommandAliasesPermission
/tardiscalltcalltardis.use
Request a player to bring their TARDIS to your targeted location.
/tardiscall [player]

/tardissay​

Send translated text in chat. For more information see TARDIS Universal Translator.

CommandAliasesPermission
/tardissaytsaytardis.translate
Say something in a different language via the TARDIS Universal Translator.
/tardissay [language] [message]

/tardisbook​

Get and list TARDIS achievement books. For more information see Timelore books.

CommandAliasesPermission
/tardisbooktbooktardis.book
Get and read TARDIS books and start TARDIS achievements.
/tardisbook [list|book name] [get|start]

/tardisgamemode​

Set a player's gamemode - if no player is specified, then the gamemode is changed for the player running the command.

CommandAliasesPermission
/tardisgamemodetgm, tgms, tgmc, tgma, tgmsptardis.admin
Set a player's gamemode
/tardisgamemode [GameMode] <player>

/tardismushroom​

Fix broken mushroom block textures.

CommandAliasesPermission
/tardismushroomtmushroomtardis.mushroom
Fix broken mushroom block textures.
/tardismushroom [red|brown|stem] [radius]

/tardisteleport​

Teleport to a world’s spawn point.

CommandAliasesPermission
/tardisteleporttardistp, ttptardis.admin
Teleport to a world’s spawn point

/tardisdisplay​

TARDIS can provide a heads up display (HUD) for players that shows various information. Activate the HUD with this command.

CommandAliasesPermission
/tardisdisplaytdisplaytardis.display
Toggle the TARDIS HUD on and off.
/tardisdisplay [ALL|BIOME|COORDS|DIRECTION|TARGET_BLOCK]

/tardistime​

Change the time in the world the player is in (or where their TARDIS is if they are inside it).

CommandAliasesPermission
/tardistimettimetardis.admin
Change the time in the world the player is in (or where their TARDIS is if they are inside it).
/tardistime [day|morning|noon|night|midnight|?AM|?PM|ticks]

/tardisweather​

Change the weather in the world the player is in (or where their TARDIS is if they are inside it).

CommandAliasesPermission
/tardisweathertweathertardis.use
Change the weather in the world the player is in (or where their TARDIS is if they are inside it).
/tardisweather [clear|rain|thunder]

You can also add a weather button to your TARDIS console to open a weather GUI. Palce an appropriate block and then run the /tardis update weather command.

When you open the weather GUI, you can click a button to change the weather, or initiate atmospheric excitation (make it snow around your TARDIS).
