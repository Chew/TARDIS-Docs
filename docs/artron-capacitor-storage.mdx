# Artron Capacitor Storage

The Artron Capacitor Storage block can be found in the Eye of Harmony room. It is used to store [Artron Capacitors](artron-capacitor) which allow you to overcharge the TARDIS with energy.

![Artron Capacitor Storage block](/images/docs/eye_storage.png)
