# TARDIS Interior SFX

You can listen to looping ambient interior sounds in your TARDIS.

:::tip

- This feature requires the [TARDIS Resource Pack](resource-packs) and the SFX player preference set to on.
- Sounds are triggered by entering the TARDIS.

:::

## Setting your interior hum preference

Use the sonic prefs menu to access the TARDIS Interior Sounds menu:

- If you want to listen to the sounds, set the _Action_ button to __Play__
- To save your sound preference, set the _Action_ button to __Save__
- Click the SFX button of your choice

![TARDIS Hum GUI](/images/docs/tardis-hum.jpg)

## Permission

`tardis.prefs.hum`

## Command

`/tardisprefs hum [sound]`
