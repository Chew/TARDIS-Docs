# TARDIS Interiors

View equirectangular panoramas of all the TARDIS interiors.

:::tip

If `desktop.previews` is `true` in _config.yml_, you can also preview interiors from the Desktop Theme GUI - shift-left-click a seed block to be transmatted to a preview interior.

:::
