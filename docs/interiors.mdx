# TARDIS Interiors

These interiors are designed to be used with the TARDIS.
To find their corresponding block required to use them, see [TARDIS types](creating-a-tardis#tardis-types).

## Sizes

Interiors come in different sizes:

| Size      | Block Dimensions | Chunk Dimensions |
| --------- | ---------------- | ---------------- |
| `SMALL`   | 16 x 16 x 16     | 1 x 1 x 1        |
| `MEDIUM`  | 32 x 32 x 16     | 2 x 2 x 1        |
| `TALL`    | 32 x 32 x 32     | 2 x 2 x 2        |
| `MASSIVE` | 48 x 48 x 32     | 3 x 3 x 2        |
