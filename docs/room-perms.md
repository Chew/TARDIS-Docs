# Room permissions

## ARS

To use the Architectural Reconfiguration System players will need:

```
tardis.architectural
```

Not to be confused with the ARCHITECTURAL room permission shown below.

## Rooms

To give permissions for all rooms, _including those added to the plugin in the future_, use:

```
tardis.room
```

Otherwise you can give individual room permissions:
