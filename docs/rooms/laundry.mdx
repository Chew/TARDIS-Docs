# Laundry Room

The laundry room contains a functional washing machine.

<TabbedImage name={"Laundry"} description={""} folder={"rooms"} image={"laundry.jpg"} />

## Washing machine

![washing machine](/images/docs/washing-machine.png)

To access the washing machine GUI, right-click the left hand quartz block in the central laundry console.

![washing machine gui](/images/docs/washing-machine-gui.jpg)

The washing machine lets you:

- Remove and return trims from armour.
- Remove colour from dyed items (leather armour, wool & carpets).

To use the washing machine:

- Place items you want to launder in the first row.
- Click the **Wash** button to remove armour trims.
- Click the **Bleach** button to remove dye colours.
- Remove your washed / bleached items from the GUI.

![washing machine gui](/images/docs/washing-machine-washed.jpg)

:::tip[Moving the washing machine]

If you want to move the washing machine to a new location, you can use the update command:

`/tardis update washing-machine`

:::
