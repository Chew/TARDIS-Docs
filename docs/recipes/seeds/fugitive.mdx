## Configuration

Depending on the plugin configuration (`creation.seed_block.legacy: false`) and the server/world difficulty, the [Redstone Torch](https://minecraft.wiki/w/Redstone_Torch) Torch item may be replaced by:

- `peaceful` / `easy` difficulty: no change - [Redstone Torch](https://minecraft.wiki/w/Redstone_Torch)
- `normal` difficulty: [Soul Lantern](https://minecraft.wiki/w/Soul_Lantern)
- `hard` / `hardcore`: [Dragon Breath](https://minecraft.wiki/w/Dragon_Breath)

:::tip

The above are the defaults for the difficulty level, but the configuration can be set by the server admin using the commands:

```
/tconfig seed_block.easy [Material]
/tconfig seed_block.normal [Material]
/tconfig seed_block.hard [Material]
```

To keep the the [Redstone Torch](https://minecraft.wiki/w/Redstone_Torch) Torch item as a Redstone Torch regardless of the server/world difficulty, set `creation.seed_block.legacy` to `true` in the config.

```
/tconfig seed_block.legacy true
```

:::
