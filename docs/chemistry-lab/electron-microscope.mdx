# Electron Microscope

:::info

The Electron Microscope requires `modules.chemistry` to be enabled in the TARDIS config.

:::

:::warning Resource Pack

All images viewed with the Electron Microscope use the [TARDIS-Resource-Pack](/resource-packs), there is not much point in using this feature without it!

:::

Electron microscopes are used to investigate the ultra structure of a wide range of biological and inorganic specimens including microorganisms, cells, large molecules, biopsy samples, metals, and crystals.

![Electron Microscope](/images/chemistry/electron-microscope.png)

To use the electron microscope, you need to craft and place it first:

`/trecipe electron-microscope`

## Usage

Right-clicking the electron microscope with a computer disk will allow you to view the sample object.

:::info

To close the electron microscope view, change another item in your hand.

:::

![Computer Disk](/images/chemistry/computer-disk.png)

Computer disks are stored in the computer.

![Computer](/images/chemistry/screen.png)

## Computer

Craft and place a computer to gain access to the electron microscope disks.

`/trecipe computer-monitor`

| Ingredients | Crafting recipe | Difficulty |
| ----------- | --------------- | ------------- |
| [Amethyst Shard](https://minecraft.wiki/w/Amethyst_Shard)<br/>[Redstone](https://minecraft.wiki/w/Redstone)<br/>[Light Blue Dye](https://minecraft.wiki/w/Light_Blue_Dye)<br/>[Glass Pane](https://minecraft.wiki/w/Glass_Pane) | <Recipe icons={['air','amethyst-shard','air','redstone','light-blue-dye','redstone','glass-pane','glass-pane','glass-pane','computer-monitor']} /> | easy |
| [Spyglass](https://minecraft.wiki/w/Spyglass)<br/>[Redstone](https://minecraft.wiki/w/Redstone)<br/>[Light Blue Dye](https://minecraft.wiki/w/Light_Blue_Dye)<br/>[Glass](https://minecraft.wiki/w/Glass) | <Recipe icons={['air','spyglass','air','redstone','light-blue-dye','redstone','glass','glass','glass','computer-monitor']} /> | hard |

:::info

To return a disk to the computer storage, right-click the computer with the disk in your hand.

:::
