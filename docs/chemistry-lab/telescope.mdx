# Telescope

:::info

The Telescope requires `modules.chemistry` to be enabled in the TARDIS config.

The Telescope and Filing Cabinet are automatically added to the Observatory room when it is grown.

:::

:::warning Resource Pack

All images viewed with the Telescope use the [TARDIS-Resource-Pack](/resource-packs), there is not much point in using this feature without it!

:::

A telescope is a device used to observe distant objects, such as observational astronomy - planets, moons, stars, nebulas and galaxies for example.

![Telescope](/images/chemistry/telescope.png)

To use the telescope, you need to craft it first:

`/trecipe telescope`

| Ingredients | Crafting recipe | Difficulty |
| ----------- | --------------- | ------------- |
| [Amethyst Shard](https://minecraft.wiki/w/Amethyst_Shard)<br/>[Redstone](https://minecraft.wiki/w/Redstone)<br/>[White Dye](https://minecraft.wiki/w/White_Dye)<br/>[Glass Pane](https://minecraft.wiki/w/Glass_Pane) | <Recipe icons={['air','amethyst-shard','air','redstone','white-dye','redstone','glass-pane','glass-pane','glass-pane','telescope']} /> | easy |
| [Spyglass](https://minecraft.wiki/w/Spyglass)<br/>[Redstone](https://minecraft.wiki/w/Redstone)<br/>[White Dye](https://minecraft.wiki/w/White_Dye)<br/>[Glass](https://minecraft.wiki/w/Glass) | <Recipe icons={['air','spyglass','air','redstone','white-dye','redstone','glass','glass','glass','telescope']} /> | hard |

## Usage

Right-clicking the telescope with a folder will allow you to view the astronomical object.

![Space helmet](/images/chemistry/folder.png)

Astronomical folders are stored in the filing cabinet.

![Filing cabinet](/images/chemistry/filing-cabinet.png)

## Filing cabinet

`/trecipe filing-cabinet`

| Ingredients | Crafting recipe | Difficulty |
| ----------- | --------------- | ------------- |
| [Amethyst Shard](https://minecraft.wiki/w/Amethyst_Shard)<br/>[Redstone](https://minecraft.wiki/w/Redstone)<br/>[Orange Dye](https://minecraft.wiki/w/Orange_Dye)<br/>[Glass Pane](https://minecraft.wiki/w/Glass_Pane) | <Recipe icons={['air','amethyst-shard','air','redstone','orange-dye','redstone','glass-pane','glass-pane','glass-pane','filing-cabinet']} /> | easy |
| [Spyglass](https://minecraft.wiki/w/Spyglass)<br/>[Redstone](https://minecraft.wiki/w/Redstone)<br/>[Orange Dye](https://minecraft.wiki/w/Orange_Dye)<br/>[Glass](https://minecraft.wiki/w/Glass) | <Recipe icons={['air','spyglass','air','redstone','orange-dye','redstone','glass','glass','glass','filing-cabinet']} /> | hard |

:::info

To return a folder to the filing cabinet, right-click the filing cabinet with the folder in your hand.

:::
