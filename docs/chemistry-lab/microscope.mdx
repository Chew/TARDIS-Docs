# Microscope

:::info

The Microscope requires `modules.chemistry` to be enabled in the TARDIS config.

The Microscope and Slide Rack are automatically added to the Observatory room when it is grown.

:::

:::warning Resource Pack

All images viewed with the Microscope use the [TARDIS-Resource-Pack](/resource-packs), there is not much point in using this feature without it!

:::

A microscope is a laboratory instrument used to examine objects that are too small to be seen by the naked eye.

![Microscope](/images/chemistry/microscope.png)

To use the microscope, you need to craft and place it first:

`/trecipe microscope`

| Ingredients | Crafting recipe | Difficulty |
| ----------- | --------------- | ------------- |
| [Amethyst Shard](https://minecraft.wiki/w/Amethyst_Shard)<br/>[Redstone](https://minecraft.wiki/w/Redstone)<br/>[Light Gray Dye](https://minecraft.wiki/w/Light_Gray_Dye)<br/>[Glass Pane](https://minecraft.wiki/w/Glass_Pane) | <Recipe icons={['air','amethyst-shard','air','redstone','light-gray-dye','redstone','glass-pane','glass-pane','glass-pane','microscope']} /> | easy |
| [Spyglass](https://minecraft.wiki/w/Spyglass)<br/>[Redstone](https://minecraft.wiki/w/Redstone)<br/>[Light Gray Dye](https://minecraft.wiki/w/Light_Gray_Dye)<br/>[Glass](https://minecraft.wiki/w/Glass) | <Recipe icons={['air','spyglass','air','redstone','light-gray-dye','redstone','glass','glass','glass','microscope']} /> | hard |

## Usage

Right-clicking the microscope with a slide will allow you to view the invisible to the naked eye object.

:::info

To close the microscope view, change another item in your hand.

:::

![Slide](/images/chemistry/slide.png)

Slides are stored in the slide rack.

![Slide rack](/images/chemistry/slide-rack.png)

## Slide rack

Craft and place a slide rack to gain access to the microscope slides.

`/trecipe slide-rack`

| Ingredients | Crafting recipe | Difficulty |
| ----------- | --------------- | ------------- |
| [Amethyst Shard](https://minecraft.wiki/w/Amethyst_Shard)<br/>[Redstone](https://minecraft.wiki/w/Redstone)<br/>[Brown Dye](https://minecraft.wiki/w/Brown_Dye)<br/>[Glass Pane](https://minecraft.wiki/w/Glass_Pane) | <Recipe icons={['air','amethyst-shard','air','redstone','brown-dye','redstone','glass-pane','glass-pane','glass-pane','slide-rack']} /> | easy |
| [Spyglass](https://minecraft.wiki/w/Spyglass)<br/>[Redstone](https://minecraft.wiki/w/Redstone)<br/>[Brown Dye](https://minecraft.wiki/w/Brown_Dye)<br/>[Glass](https://minecraft.wiki/w/Glass) | <Recipe icons={['air','spyglass','air','redstone','brown-dye','redstone','glass','glass','glass','slide-rack']} /> | hard |

Right-click the slide rack to open the slides GUI

:::info

To return a slide to the slide rack, right-click the slide rack with the slide in your hand.

:::
