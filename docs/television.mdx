# TARDIS Television

The TARDIS Television is used to select and apply Doctor Who themed skins to players.

![TARDIS Television](/images/skins/tv.png)

Place a television block then run the `/tardis update television` command to install it in your TARDIS.

Right-click the block to open the skins GUI - for more information, visit the [Skins](skins) page.
